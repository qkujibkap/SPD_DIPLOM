VKinder — Бот для поиска знакомств во ВКонтакте

VKinder — чат-бот для платформы ВКонтакте, который подбирает пользователей для знакомств на основе данных самого пользователя (возраст, пол, город), показывает карточки людей с фото и позволяет сохранять понравившихся в избранное.

------------------------------------------------------------
Возможности бота
------------------------------------------------------------

- Определение пола, возраста и города пользователя
- Поиск подходящих людей через VK API
- Отправка карточек:
    * Имя и фамилия
    * Ссылка на профиль
    * Топ-3 фотографии по лайкам
- Работа с избранным:
    * Добавление пользователя
    * Просмотр списка избранных
- Команды бота:
    * Начать
    * Следующий
    * В избранное
    * Показать избранных
    * Стоп

------------------------------------------------------------
Настройка переменных окружения
------------------------------------------------------------

Файл config.py не содержит токенов. Все токены задаются в системе.

macOS / Linux:

    export TOKEN_GROUP="ваш_токен_группы"
    export TOKEN_USER="ваш_user_token"
    export GROUP_ID=123456789

Windows PowerShell:

    setx TOKEN_GROUP "ваш_токен_группы"
    setx TOKEN_USER "ваш_user_token"
    setx GROUP_ID "123456789"

Проверка:

    echo $TOKEN_GROUP
    echo $TOKEN_USER
    echo $GROUP_ID

------------------------------------------------------------
Запуск проекта
------------------------------------------------------------

1. Установить зависимости:

       pip install -r requirements.txt

2. Убедиться, что переменные окружения установлены.

3. Запустить бота:

       python bot.py

------------------------------------------------------------
Описание модулей
------------------------------------------------------------

favorites.py:
    - load_favorites() — загрузка списка избранных
    - save_favorites() — сохранение списка
    - add_to_favorites() — добавление пользователя
    - get_favorites() — получение списка

vk_client.py:
    - search_users() — поиск людей (пол, возраст, город)
    - get_top_photos() — получение топ-3 фото

bot.py:
    - обработка команд
    - получение данных о пользователе
    - выбор подходящих людей
    - отправка карточек
    - работа с избранным
